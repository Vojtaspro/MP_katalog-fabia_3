# 🏎️ Srovnávač Škoda Fabia (Maturitní projekt)

Tato aplikace slouží k detailnímu porovnání technických parametrů různých generací a motorizací vozu **Škoda Fabia**. Projekt byl vytvořen jako součást maturitní práce.

## 🚀 Funkce
* **Interaktivní výběr:** Možnost zvolit dvě různé motorizace pro přímé porovnání.
* **Vizuální srovnání:** Barevné odlišení lepších a horších parametrů (zelená/červená).
* **Technické specifikace:** Porovnání výkonu, točivého momentu, zrychlení, spotřeby a dalších údajů.
* **Celkový vítěz:** Automatický výpočet vítěze na základě zvolených kritérií.

## 🛠️ Použité technologie
* **Python** (hlavní programovací jazyk)
* **Streamlit** (webové rozhraní a dashboard)
* **Pandas** (zpracování dat z Excelu)
* **Plotly** (vizualizace dat)

## 📦 Instalace a spuštění
Pokud si chcete aplikaci spustit lokálně:
1. Nainstalujte potřebné knihovny: `pip install -r requirements.txt`
2. Spusťte aplikaci příkazem: `streamlit run app_3.py`
